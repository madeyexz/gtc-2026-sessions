## Abstract
As AI models scale, data center networking must provide ultra-low latency, high throughput, and programmable data paths so training and inference workloads remain efficient. With the NVIDIA DOCA software platform, developers can move critical data-path functions onto NVIDIA BlueField data processing units (DPUs) using DOCA libraries and microservices to offload, accelerate, and isolate cloud infrastructure while enhancing observability. We’ll cover how DOCA enables centralized orchestration of BlueField‑accelerated services, dynamic traffic steering, and fine‑grained network and security controls that work together to deliver secure, high‑performance fabrics with predictable performance and efficient resource utilization for multi‑tenant AI environments at scale.
