## Abstract
Learn how to eliminate communication bottlenecks with NVIDIA Communication stack. Over the past year, each library has evolved to address specific bottlenecks: NCCL introduced Symmetric Memory support and Copy Engine collectives, enabling zero-SM communication. NVSHMEM enhanced GPU-initiated operations in the Python ecosystem. NIXL emerged as the inference-focused solution, providing async, noncontiguous transfers across memory tiers (GPU HBM, CPU DRAM, local/networked storage) with backends including GDS, UCX, and S3—essential for disaggregated inference architectures.

Explore developer optimizations: custom collectives in NCCL, KV cache and storage innovations in NIXL, and expert parallelism patterns. Learn how Spectrum-X plugins extend capabilities across multi-site deployments for scale-across connectivity.
