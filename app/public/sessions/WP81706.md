Building agents is easy; reliably improving them is hard. As workflows shift to complex long-horizon tasks, traditional testing fails. How do you debug and improve non-deterministic behavior?

This session presents a framework for evaluation-driven development. We move beyond “vibe eval” to disciplined engineering covering ablations, unit tests, trajectory evaluation, and LLM-as-judge patterns. Using a case study of achieving No. 1 on a Deep Research leaderboard, we demonstrate how to build pipelines that catch hallucinations and regressions while preventing overfitting. You'll leave with a workflow for implementing evaluation methods that take agents from prototype to scalable deployment using a suite of tooling enabled by NVIDIA NeMo and open NVIDIA Nemotron models for advanced reasoning.

Building agents is easy; reliably improving them is hard—especially for long-horizon, non-deterministic tasks where traditional testing fails. This session introduces a disciplined framework for evaluation-driven development, moving beyond “vibe eval” to structured methods like ablations, unit tests, trajectory evaluation, and LLM-as-judge patterns.
Using a case study of achieving No. 1 on a Deep Research leaderboard, we demonstrate pipelines that catch hallucinations and regressions while preventing overfitting. Learn how to implement evaluation workflows that scale agents from prototype to production.
The workflow leverages NVIDIA NeMo Evaluator for reproducible benchmarking, NeMo Guardrails for safety, NeMo Agent Toolkit for orchestration, and NVIDIA Nemotron models for advanced reasoning. We’ll show how these tools integrate to enable robust, scalable agent development.
