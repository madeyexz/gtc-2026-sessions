## Speakers
### Maximo Guk (Presenter)
> Maximo is a software engineer on Cloudflare's Developer Platform working on edge computing and AI-related products (e.g. MCP).

## Abstract
Cloudflare provides a serverless platform for building agents that can run code, execute tools, and persist state without managing servers or backend infrastructure. In this session, we’ll show how workers, durable objects, and Cloudflare’s Agents SDK can power a self-learning agent by having the agent control a car to go around obstacles in a simulated environment. This illustrates how Cloudflare enables agents that can learn, adapt, and evolve over time, rather than operate as stateless chat interfaces.
