## Abstract
Discover how TornadoVM opens the NVIDIA GPU ecosystem to millions of Java developers, a community where GPU adoption has traditionally been extremely low. We demonstrate how TornadoVM’s hybrid API enables pure Java code to compile to PTX and interoperate with NVIDIA libraries like cuDNN, bridging GPU compute with the broader JVM ecosystem through frameworks such as GraalVM, LangChain4j, and Quarkus.

As an open-source framework, TornadoVM provides a seamless bridge between Java applications and NVIDIA’s developer tooling, allowing profiling, debugging, and optimization of GPU kernels directly from Java. This work highlights how GPU acceleration can become a first-class capability for the JVM community, unlocking new opportunities for AI and high-performance computing in pure modern Java.
