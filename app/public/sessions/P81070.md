## Abstract
This work introduces the first open-source framework for benchmarking federated learning (FL) under dynamic client participation, demonstrating that this practical scenario causes significant performance degradation. To address this challenge, we propose knowledge-pool federated learning (KPFL), a generic plugin that uses a shared knowledge pool and generative knowledge distillation to improve model robustness and prevent knowledge loss.
