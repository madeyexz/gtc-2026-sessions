## Abstract
Learn how to build robust OpenUSD-based data pipelines using USD Exchange SDK, to leverage existing 3D software and tools for scalable physical AI workflows. Start from common 3D source formats and write Python-based code to extract, transform, and load assets into reusable, modular USD representations suitable for large-scale workflows.
Through guided, hands-on exercises, structure 3D content into composable USD assets, apply simulation-ready (“SimReady”) standards, and enrich assets with rigid-body physics that support downstream robotics and simulation workloads. By the end, you'll have automated an asset conversion pipeline capable of processing large 3D datasets, for a practical blueprint to adapt to own production environment. 

Prerequisite(s):


Intermediate experience with Python: working with packages, virtual environments, basic scripting.


Familiarity with content concepts: meshes, materials, transforms, scene hierarchies common to 3D modeling applications.


Understand command-line usage for running scripts, managing project files.


Recommended:


Completed Learn OpenUSD curriculum, covering fundamentals of USD-based scene representation and concepts (stages, layers, prims, composition).


Basic familiarity with robotics, simulation or general 3D content workflows (simulation tools, physics-enabled assets).
