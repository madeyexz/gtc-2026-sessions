## Abstract
LLM 作为 AI 技术的核心分支，正在重塑量化投资范式。LLM 具有超乎寻常的语义理解与推理能力，可以作为强大的工具模型嵌入量化投资流程。
从 LLM 的输出随机性出发，我们可以依赖 LLM 进行因子挖掘，并基于 NVIDIA NIM 和 TensorRT-LLM 加速挖掘；
从 LLM 的语义理解能力出发，我们基于 GPU 本地部署LLM解读海量文本、提取情感信息，构建或优化量化选股策略；
从 LLM 的多模态思维链出发，我们可利用 LLM 对K线图像进行交互式技术分析。LLM 正在引发一场量化投资技术更迭的革命......
