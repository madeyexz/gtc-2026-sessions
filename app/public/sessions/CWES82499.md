## Abstract
Running today’s self-evolving AI agents is fundamentally insecure. Agents such as OpenClaw operate with real system access but lack a robust, infrastructure-level security model. Tool-level restrictions can be bypassed when agents discover alternate execution paths, and container or prompt-based guardrails are insufficient for long-running systems. In this session, experts will discuss infrastructure-level security and runtime platforms for deploying autonomous agents more securely by default.
