## Abstract
We present a GPU-accelerated real-time voice interaction system built on NVIDIA NeMo, TensorRT-LLM, and Riva, capable of running directly over standard SIP/RTP telephony. Using only one RTX 5070 Ti or RTX 5090 GPU, the system achieves sub-600 ms end-to-end conversational latency, performing automatic speech recognition (ASR), LLM inference, and text-to-speech (TTS) fully on-device. This work demonstrates how small and medium-sized enterprises (SMEs) can deploy on-premises AI voice agents using consumer-grade GPUs, reducing cloud costs and ensuring secure, low-latency communication.
