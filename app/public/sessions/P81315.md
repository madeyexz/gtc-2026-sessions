## Abstract
Thousands of inquiry tickets received annually by IT support can be resolved using existing knowledge bases but require manual effort, prolonging resolution times. The Auto Help System automates this process using an LLM-powered architecture with intelligent classification agent and a context-aware, retrieval augmented generation (RAG)-based generator that crafts human-like, accurate responses. A multi-layered guardrail ensures safe and relevant outputs, while an automated workflow updates tickets, notifies users via Slack and email, and escalates when needed. We achieved 89% accuracy where the system delivers a median time savings of 39 minutes per ticket, and on average two days of fulfillment time for customers, and is projected to self-resolve over 8,000 tickets annually, resulting in an estimated $880,000 in savings while enhancing user satisfaction and operational efficiency.
