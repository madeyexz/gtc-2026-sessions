## Abstract
In this training lab, we'll introduce Project Aether — a set of automation and AI tools that speed up and reduce the cost of accelerated extract, transform, load and analytics workloads on the RAPIDS Accelerator for Apache Spark, built on NVIDIA cuDF. Aether combines automation along with advanced AI tools to automatically test and optimize your Apache Spark workloads with GPU.
 
Prerequisite(s):

Some experience with Linux and the command line needed.
An understanding of large-scale analytics and Apache Spark is not required, but would be helpful.
