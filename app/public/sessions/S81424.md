## Abstract
Unpredictable workloads make inference a natural fit for serverless computing (a.k.a. "functions-as-a-service"). But serverless execution only works if new function replicas spin up quickly. Learn the techniques we've developed at Modal to speed up the boots of GPU containers and achieve truly serverless execution for workloads running on thousands of GPUs.
