## Abstract
Static prompt engineering breaks under real-world complexity. This session shows how Reinforcement Learning (RL) enables agents to make better decisions, use tools reliably, and adapt in production.

This talk distills lessons from Atlassian on how it built Rovo, its enterprise AI solution for search, AI chat, and agents. The team explains how it moved from static prompt engineering to policy training via RL thereby improving intent interpretation, retrieval quality and how they use NVIDIA NeMo RL to evolve query rewriting from brittle rules to adaptive learning.

The session then expands to platform RL, where Google Cloud Vertex Training Cluster showcases how its integration of NVIDIA NeMo libraries has helped customers build best-in-class models. The team will deep dive on Vertex’s managed GPU infrastructure and how NeMo RL and other techniques can be used to build high performance agents."
