## Abstract
Explore the latest advancements in GPU-accelerated Mixed Integer Programming (MIP) solver, generating fast, high-quality feasible solutions, which is a critical capability for dynamic decision making. You’ll learn about the algorithms behind the cuOpt Linear Programming (LP) solver, which earned the top spot as an open source solver on the Mittelman LPfeas benchmark. 
We’ll explore how Nemo Agent Toolkit and cuOpt optimization are tackling some of today’s most complex real-world decision problems and share our latest work on a modular skill-based decision intelligence agent that enables users to generate optimized decisions and plans through natural language.

Additionally, you'll learn how Kinaxis achieved speedups of up to 14X on key supply chain planning use cases by integrating NVIDIA cuOpt Linear Programming (LP)​ on NVIDIA GPUs into the Maestro platform for scenario planning.
