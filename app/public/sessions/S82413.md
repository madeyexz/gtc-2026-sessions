## Abstract
Learn how an AI system can evolve from simple search into long-horizon agents that run for hours or days to complete complex, real-world tasks. This session walks through four stages of that evolution—AI search, asynchronous workflows, agents, and long-horizon agents—and examines the architectural patterns and technical trade-offs required at each step. We will cover practical approaches we learned building Genspark, including model routing, tool execution frameworks, validation of agent trajectories, and reliability strategies needed to operate agent systems at scale.
