## Abstract
Data engineering can dramatically benefit from agentic assistance. However, agents are not well suited to many standard data integration tasks, as hallucinations during pipeline execution can produce non-deterministic outcomes, and operations rarely merit the expense of AI at scale. We demonstrate a system that leverages agents to help design abstract data pipelines as Directed Acyclic Graphs (DAGs), then translate those pipelines into code that can be deployed in existing data platforms. This ensures deterministic, traceable, and affordable data workflows, while still benefiting from the huge advantages agents offer in accelerating building pipelines. A combination graph and chat based user interface is demonstrated allowing efficient human-machine teaming for pipeline build-out.
